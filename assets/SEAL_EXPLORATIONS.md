# Seal Explorations Archive

These generated studies explored a seal-like identity for TravelPins. They are
kept for visual research only and are not production assets.

The direction was set aside because the vessel and cloud forms felt too literal,
too icon-like, or disconnected from the quieter handscroll-inspired interface.
The product now uses a typographic wordmark, landscape pacing, and restrained
mineral color roles instead of a standalone cultural symbol.

Files in this archive:

- `travelpins-seal-cloud-source.png`
- `travelpins-seal-cloud.png`
- `travelpins-seal-final.png`
- `travelpins-seal-v1.png`
- `travelpins-tea-seal-source.png`

Do not merge these files into `main` without a new identity review.
